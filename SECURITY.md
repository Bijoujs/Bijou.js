# Security Policy
We try to keep stuff secure. Make a PR if you want to fix something. I do my best.
## Supported Versions


| Version | Supported          |
| ------- | ------------------ |
| > 2.0.0  | :white_check_mark:                |
| < 2.0.0   | :x:                |

## Reporting a Vulnerability

Make a new issue and use the vulnerability template.
